KMT (KMT Manipulates Text) is a really bad text editor.

Made by Bryce Walther (bryce@thewalthers.com) with assistance from Edward Lee

Goals for this project :
Make a terminal-based editor with ncurses (Run from the command line)
Commands: Save, Close, Copy, Paste, Open
		
This project is for learning purposes, and made for and on linux.

If you encounter any bugs, please send reports to bryce@thewalthers.com

Hope you enjoy!
